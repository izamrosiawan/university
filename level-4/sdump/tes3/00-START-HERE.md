## 🎯 Cara Upload (3 Pilihan)

### Pilihan 1: FTP Upload (Paling Aman)
1. Connect FTP ke hosting
2. Upload folder `tes3` ke `/wp-content/themes/`
3. Activate di Dashboard → Appearance → Themes

### Pilihan 2: WordPress Dashboard Upload
1. Compress `tes3` menjadi `tes3.zip`
2. Dashboard → Appearance → Themes → Add New
3. Upload ZIP file
4. Activate

### Pilihan 3: cPanel File Manager
1. Login cPanel
2. File Manager → wp-content/themes/
3. Upload folder tes3
4. Activate di WordPress

**Lihat file `WORDPRESS_UPLOAD_GUIDE.md` untuk detail lengkap!**
