## 🚀 HOW TO UPLOAD TO WORDPRESS

### Easy Steps:

**Method 1: WordPress Dashboard (Simplest)**
```
1. Go to Dashboard → Appearance → Themes
2. Click "Add New"
3. Click "Upload Theme"
4. Select tes3.zip file (compress folder first)
5. Click "Install Now"
6. Click "Activate"
```

**Method 2: FTP Upload (Most Reliable)**
```
1. Connect FTP to your hosting
2. Navigate to /wp-content/themes/
3. Upload tes3 folder
4. Go to WordPress Dashboard → Appearance → Themes
5. Find "SDN Wonokromo III" and click "Activate"
```

**Method 3: cPanel File Manager**
```
1. Login to cPanel
2. File Manager
3. Navigate to public_html/wp-content/themes/
4. Upload tes3 folder
5. Activate from WordPress
```

## ⚙️ AFTER ACTIVATION

Once activated, you need to:

1. **Upload Logo**
   - Dashboard → Appearance → Customize → Site Identity

2. **Create Navigation Menu**
   - Dashboard → Appearance → Menus
   - Create menu dengan items: Home, Profil, Galeri, Berita, Kontak
   - Set to "Primary Menu"

3. **Set Homepage**
   - Dashboard → Settings → Reading
   - Select "A static page"
   - Set Homepage & Posts page

4. **Fill Theme Settings**
   - Dashboard → Appearance → Customize
   - Fill Hero title, Welcome message, Footer info

5. **Create Pages & Posts**
   - Create pages: Profil, Galeri, Kontak
   - Create sample posts/berita
