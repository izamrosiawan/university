# Adventure Game - Kalahkan Monster

Game petualangan sederhana berbasis teks. Tugasmu adalah mengalahkan semua monster yang menghadang!

## Cara Main

1. Pastikan Python 3 sudah terpasang.
2. Jalankan game dengan perintah:
   ```
   python main.py
   ```
3. Ikuti instruksi di layar:
   - Pilih nama pahlawan.
   - Pilih aksi di setiap giliran: serang (s) atau sembuh (h).
   - Kalahkan semua monster untuk menang!

Selamat bermain!
